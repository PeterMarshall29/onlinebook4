Identifiers#

According to cplusplus.com

A valid identifier is a sequence of one or more letters, digits, or underscore characters (). Spaces, punctuation marks, and symbols cannot be part of an identifier. In addition, identifiers shall always begin with a letter. They can also begin with an underlined character (), but such identifiers are -on most cases- considered reserved for compiler-specific keywords or external identifiers, as well as identifiers containing two successive underscore characters anywhere. In no case can they begin with a digit.

Valid identifiers in C++ interpreters include

alignas, alignof, and, and_eq, asm, auto, bitand, bitor, bool, break, case, catch, char, char16_t, char32_t, class, compl, const, constexpr, const_cast, continue, decltype, default, delete, do, double, dynamic_cast, else, enum, explicit, export, extern, false, float, for, friend, goto, if, inline, int, long, mutable, namespace, new, noexcept, not, not_eq, nullptr, operator, or, or_eq, private, protected, public, register, reinterpret_cast, return, short, signed, sizeof, static, static_assert, static_cast, struct, switch, template, this, thread_local, throw, true, try, typedef, typeid, typename, union, unsigned, using, virtual, void, volatile, wchar_t, while, xor, xor_eq

Very important: The C++ language is a case sensitive language.

Declaration and Initialization of variables#

C++ is a strongly-typed language and requires every variable to be declared with its type before its first use. This informs the compiler of the size to reserve in memory for the variable and how to interpret its value.

Types of initialize a variable#

In C++, there are 3 different types to declare/initialize a variable:

  1. C-like initialization

int x = 5;

  1. Constructor initialization

int x (0);

  1. Uniform initialization

int x {0};

int a = 5; //C-like initialization
int b (3); //Constructor initialization
int c {2}; //Uniform initialization
std::cout << a << " " << b << " " << c << std::endl;

5 3 2

string in C++#

string in C++ can only be declared by using std and #include <iostream>

std::string sample = "This is a string";

You can make this even shorter by using using namespace std;, but I won’t recommend.

std::string sample = "This is a string";
std::cout << sample << std::endl;

This is a string

Declare multiple variables#

In C++, you can declare multiple variables in types.

int x, b; //Declare empty variables
x = 15;
b = 1031;
int y = 2, c = 4; //Declare where the value of y is 2, and c is 4
std::string z, a = "5"; //Declare where z is the empty variable, and a is 5

Note: you cannot declare multiple variable types

int x, string b; //Error then

int x, b; //Declare empty variables
x = 15;
b = 1031;
int y = 2, c = 4; //Declare where the value of y is 2, and c is 4
std::string z, a = "5"; //Declare where z is the empty variable, and a is 5
std::cout << "Value of x is: " << x << std::endl;
std::cout << "Value of y is: " << y << std::endl;
std::cout << "Value of z is: " << z << std::endl;
std::cout << "Value of a is: " << a << std::endl;
std::cout << "Value of b is: " << b << std::endl;
std::cout << "Value of c is: " << c << std::endl;
